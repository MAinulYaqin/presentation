React Native

Apa itu React Native ?


React Native adalah Framework Javascript yang dibuat oleh Facebook dan digunakan developer untuk membuat aplikasi mobile menggunakan javascript.

  5. A ton of packages on npm

Cross Platform

Write once, run everywhere

Dengan react native, para developer cukup menuliskan kode sekali untuk Ios dan android yang akan dicompile ke Java / Objective-C

Truly Native

Kebanyakan mobile apps yang dibangun dengan javascript menggunakan cordova / ionic hanyalah sebuah webView, berbeda dengan React Native yang dicompile ke dalam bahasa native

Friendly for web developer

Sebenarnya, React Native memang ditujukan untuk para web developer yang ingin membuat mobile app dengan javascript.
